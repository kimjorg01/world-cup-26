World Cup App - Arena/Bracket Refinement

Changes in this version:
- Wider app layout using more of the screen, while keeping a centered max width.
- Arena characters now wander around again like the original version.
- Ranking is now shown through character size, leader spotlight/crown, last-place raincloud, and the ranking cards below.
- Added form status labels/emojis on player cards and above characters.
- Crowd is divided into clear left-to-right player sections based on each player's share of total points.
- Bracket labels are now English: Round of 32, Round of 16, Quarter-finals, Semi-finals, Third-place Match, Final.
- Rocket-style bracket tree is compressed to fit the page without sideways scrolling on normal desktop widths.
- Full bracket round grid was adjusted to fit the wider layout better.
- Bronze/third-place finalists now auto-fill from the semi-final losers if the bronze slots are missing.

Note:
- On small screens, the bracket will still allow horizontal scrolling because there simply is not enough width.
